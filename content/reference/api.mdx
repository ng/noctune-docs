# API Reference

Coming soon. Noctune will expose a public HTTP API for programmatic access to encounters, transcripts, and SOAP notes.

## What's planned

- **Read-only encounter and transcript endpoints** — pull SOAP notes into PIMS integrations
- **Upload endpoint** — submit audio from a third-party recorder
- **Webhooks** — get notified when an encounter finishes processing
- **OAuth 2.0** — third-party apps acting on behalf of a vet

## Interested?

If you're building an integration or want early access to the API, contact support and tell us what you're building.
