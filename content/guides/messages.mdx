# Messages

Messages is a vet-to-vet inbox for referrals, questions, and follow-ups that span multiple encounters. The sidebar shows an unread badge when you have new messages.

![Messages inbox with thread list on the left and active thread on the right](https://placecats.com/960/540)

## Threads

Each thread groups messages between you and another vet (or a whole team). Replying stays in the same thread — there's no separate "reply" vs "new message" flow once a conversation is going.

![Message thread showing alternating messages between two vets with timestamps](https://placecats.com/900/520)

## Starting a new message

Click **New Message** in the top right. Pick a recipient, optionally attach an encounter for context, and write your note.

![New message composer with recipient picker, encounter attachment, and body field](https://placecats.com/900/480)

> **Tip:** Attaching an encounter links the full SOAP note into the thread — useful for referrals where the receiving vet needs context before the patient arrives.

## Notifications

You get an email for new messages if you have that enabled under [Notifications settings](/guides/settings/notifications). The in-app badge updates in real time.

![Sidebar showing the Messages item with an unread count badge](https://placecats.com/600/240)
