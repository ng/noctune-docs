# Email Templates

Email Templates are reusable subjects and messages for discharge instructions, follow-ups, reminders, and other client communication. They are separate from [SOAP templates](/templates), which control how encounter notes are generated.

![Email Templates library with category filters, search, sorting, and a template preview](https://placecats.com/960/540)

## Find and organize templates

The library groups templates by category. Search names, subjects, and message text; filter to a category or **Favorites**; and sort by **Most used**, **Recently used**, or **A–Z**. Select the star on a template to keep it in Favorites.

Each team member has their own favorites. Usage is recorded when a message is sent, so abandoned drafts do not affect the most-used and recently-used order.

## Create or edit a template

Choose **New Template**, then add a name, optional category, subject, and body. The body supports markdown, including bold, italics, and bulleted lists; the toolbar and common keyboard shortcuts add the appropriate markdown syntax.

Use **Write** to edit and **Preview** to see a sample message. You can also duplicate an existing template and edit the copy. Noctune warns before discarding unsaved changes.

## Add placeholders

Placeholders personalize a message when you insert it into the composer. Type `{` at the cursor to open the searchable placeholder menu, then choose a value:

| Group    | Available values               |
| -------- | ------------------------------ |
| Patient  | Patient's name, species, breed |
| You      | Your name                      |
| Practice | Practice name, phone, email    |
| Date     | Today's date                   |

The editor displays placeholders as labeled chips. Preview uses sample values so you can review the finished shape of the message; an unrecognized placeholder is highlighted in amber.

## Insert a template into a message

From the library, choose **Insert into message** to open a new composer. You can also choose **Templates** while writing a new message or reply.

When inserted:

- the template subject fills the subject only if it is empty
- the template body is appended rather than replacing your draft
- sender, practice, and date placeholders fill immediately
- patient placeholders fill when a recipient matches a patient record
- an **Undo** action restores the draft to its pre-insert state

Before sending, Noctune highlights any placeholders that could not be filled and explains why. Review them carefully. If you choose **Send anyway**, the unresolved placeholder text is included in the email.
